# Daily Google Ads Management SOP

## Purpose

Use this process to manage Google Ads like a performance agency.

## Daily Workflow

1. Run Codex Daily QA.
2. Check campaign settings.
3. Check spend.
4. Check search terms.
5. Check calls/conversions.
6. Check location quality.
7. Check disapprovals.
8. Add necessary negatives.
9. Record all changes in CHANGELOG.md.
10. Do not make strategic changes from tiny data.

## Daily Checks

| Area | Check |
|---|---|
| Campaign status | Enabled and eligible |
| Budget | Correct daily budget |
| Bidding | Approved strategy only |
| Network | Search-only |
| Keywords | No broad match |
| Search terms | No competitor/far-city/waste spend |
| Negatives | Applied correctly |
| Ads | Approved |
| Assets | Call asset correct |
| Landing page | Final URL correct |
| Phone | (425) 673-0700 |
| Tracking | Any conversion issues |

## What Can Be Changed Daily

Allowed:
- Add obvious negative keywords.
- Fix broken URL.
- Fix wrong phone number.
- Fix disapproval.
- Pause obviously dangerous keyword.

Not allowed without strategy approval:
- Change bidding.
- Increase budget.
- Add broad keywords.
- Turn on Display.
- Turn on Search Partners.
- Launch new campaign.
- Switch to Target CPA.
- Change conversion goals.

## Daily Report Format

- Spend yesterday:
- Clicks:
- Conversions:
- Search term issues:
- Negatives added:
