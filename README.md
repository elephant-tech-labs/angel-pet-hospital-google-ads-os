# Angel Pet Hospital Google Ads Operating System

This repository is the source of truth for managing Google Ads growth for Angel Pet Hospital in Edmonds, WA.

## Purpose

This repo stores:
- Google Ads strategy
- Campaign structure
- Campaign rules
- Search term reviews
- Negative keyword decisions
- Landing page briefs
- Conversion tracking plans
- Change logs
- Client reports
- Codex audit prompts
- Weekly optimization SOPs

## Main Goal

Generate qualified calls, appointment requests, online bookings, and form leads from nearby pet owners using Google Ads.

## Operating Principle

This is not a random campaign management folder. This is an agency-grade operating system.

Every recommendation must improve one of these:
- lead quality
- call volume
- booked appointments
- cost per qualified lead
- search term quality
- conversion rate
- campaign scalability

## AI Workflow

Codex with Google Ads MCP reads the live ad account.

This GitHub repo gives Codex the strategic context.

ChatGPT acts as the strategy director and decision-review layer.

Human approval is required before major changes.
